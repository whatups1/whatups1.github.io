About Me

Projects

Pixel Sprite Editor

The Pixel Sprite Editor is a tool designed for creating and editing pixel art sprites. It provides a straightforward interface for users to craft 2D art, commonly used in games, animations, etc. The editor features pixel mirroring, object fill, and multiple pen sizes.

Learning Management System

For the final project in the Database Management elective, my partner and I developed a Learning Management System (LMS) using C# with MySQL for the database backend. The LMS comprised several essential components, including tables for managing students, teachers, classes, grades, and time slots. These tables facilitated efficient organization and retrieval of data necessary for the system's functionality.

Hiking and Weather App

Using Android Studio and Kotlin, I developed an app capable of retrieving and displaying current weather information. The app fetches data from an online database and parses JSON data based on the user's location. In addition to providing weather updates, the app offers a feature to explore local hiking trails. Users can navigate to this functionality through a separate fragment, and learn about these hikes. This data was acquired through the Google Maps API.
